Domain Hosting Health Checker (Java GUI)

Overview
Domain Hosting Health Checker is a Java Swing-based GUI application that simulates monitoring the health of multiple domains.  
It allows users to view, search, update, and analyze domain hosting information such as DNS and SSL status.  

This project was built in NetBeans IDE as part of learning and applying Java programming concepts, object-oriented design, and GUI development.

---
Features
- **View All Domains**: Display hosting provider, DNS status, and SSL status for all domains.
- **Search Domain**: Search for a specific domain by name.
- **Update Domain**: Modify domain details (name, provider, DNS, SSL).
- **Count Healthy Domains**: Show the number of domains with valid DNS and SSL.
- **Analytics Summary**: Generate insights such as:
  - Total domains
  - Expired SSLs
  - DNS errors
  - Most common hosting provider
  - Flagged domains (issues with DNS or SSL)
- **Clear & Exit**: Reset the display area or close the application.
